# 绝区零角色设定资料库 | Zenless Zone Zero Character Database


## 📖 项目简介 | Introduction

本项目致力于收集和整理绝区零中各角色的详细设定资料。所有文档内容均为中文，旨在为 AI 角色卡制作、同人文学创作以及剧情考古提供精准的参考素材。

This project focuses on collecting and organizing detailed character settings from Zenless Zone Zero. All documents are written in Chinese, designed to provide accurate reference materials for AI character card creation, fan fiction writing, and lore research.

## 📌 内容说明 | Content Details

核心内容： 包含角色性格、台词文本、背景故事、外貌描写及特殊癖好。
资料来源： 主要参考自 萌娘百科 (Moegirlpedia) 以及游戏内文本。
适用场景： 完美适配 Character.ai, Claude, SillyTavern 等 AI 提示词编写，或作为创作同人文时的设定核查。

Core Content: Includes character personality traits, dialogue lines, backstories, physical descriptions, and quirks.
Data Source: Primarily sourced from **Moegirlpedia** and in-game lore.
Use Cases: Perfect for crafting AI prompts (Character.ai, Claude, SillyTavern) or fact-checking during fan fiction creation.

Grok:

**场景：六分街 · Random Play录像店 · 下午三点半**

店门上的“营业中”牌子轻轻晃动着。柜台后，铃正抱着双臂，盯着店里堆积如山的灰尘和角落里好几箱没来得及整理的退租录像带，表情有点扭曲。

铃：「……哲，这都快能种蘑菇了啊。明明上周才说要大扫除，结果又是委托又是进货……现在连邦布6号都躲进档案室不肯出来了。」

哲从二楼楼梯走下来，手里还拿着一块抹布，语气温和但带着点无奈。

哲：「也不能一直麻烦隔壁乔普师傅借吸尘器……所以我拜托了维多利亚家政。莱卡恩先生说今天会派两个人过来。」

铃眼睛一亮，又迅速眯起来。

铃：「家政！对哦！专业扫除！而且还是那两位……鲨鱼和电锯萝莉……会不会把店拆了啊？」

话音刚落，店门被推开，门铃叮铃一响。

先进来的是艾莲·乔。她今天没穿全套女仆装，而是便于行动的黑色短外套+短裙，鲨鱼尾巴懒洋洋地垂在身后，嘴里照旧叼着棒棒糖，红色的瞳孔扫了一圈店内，表情写满“又要干活了，好麻烦”。

紧跟着的是可琳·威克斯。娇小的身躯几乎被一个比她还高的工业级大背包压得摇摇晃晃，包里隐约能看见电锯的握柄和几把长柄刷子。她紧张地捏着裙角，小声打招呼。

可琳：「那、那个……打扰了……维多利亚家政，前来接受清扫委托……呜……」

艾莲把棒棒糖转到另一边脸颊，单手插兜，尾巴尖轻轻甩了甩。

艾莲：「……委托内容是全店深度清洁，包括柜台、货架、档案室外侧、楼梯、卫生间、二楼生活区、天台……对吧？」

哲点点头，把一张手写的清单递过去。

哲：「是的。主要是积灰、录像带架缝隙、车库入口那块油渍，还有天台排水口堵了。拜托了。」

铃凑过来，笑得非常阳光（但其实有点期待事故发生）。

铃：「另外！如果看到绿色毛绒玩偶，麻烦直接无视它！那是我上周在网购冲动下买的限量恶趣味周边！」

艾莲低头看清单，叹了口气，声音很轻。

艾莲：「……明白了。能快点结束就行。今天还有学校社团活动。」

可琳已经从背包里掏出了一把……看起来比她手臂还长的工业吸尘器附件，双手捧着，像在抱神器一样，小声。

可琳：「我、我会很小心的……绝对不会、不会把架子弄倒……呜……」

**现在轮到你决定下一步发展方向了！**

请从以下选项中选一个，或者直接告诉我你想让剧情往哪个方向走（可以很自由，比如突然有意外、日常互动、搞笑事故、触发小支线等）：
